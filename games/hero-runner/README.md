# 🦸 Hero Runner

바닐라 HTML5 Canvas, CSS, JavaScript로 만든 픽셀 사이드스크롤 러너 게임입니다.

## 기능

- **3종 장애물** — 각각 다른 회피 패턴을 구조적으로 강제
- **이중 점프** — 공중에서 한 번 더 점프 가능
- **슬라이드** — `↓` 키를 누르는 동안 자세 낮추기
- **코인 수집** — 장애물 유형에 연동된 위치에 스폰, 파티클 효과
- **속도 증가** — `speed = 3.2 + √score × 0.1` 공식으로 점진적 가속
- **스폰 간격 단축** — `max(38, 80 - √score × 2)` 프레임으로 밀도 증가
- **최고 기록** — `localStorage`로 영구 저장
- **픽셀 아트 렌더링** — 외부 이미지 없이 Canvas `fillRect`로 모든 스프라이트 직접 드로잉
- **CRT 스캔라인** — 2px 간격 반투명 오버레이
- **픽셀 배경** — 깜빡이는 별 + 픽셀 달 + 계단식 산 실루엣 2레이어 (별도 캔버스)
- **사이드 UI 패널** — Score / Coins / Best 실시간 표시, 조작법 및 장애물 가이드

## 장애물

| 장애물 | 회피 방법 | 이유 |
|--------|-----------|------|
| 🪨 **Rock** | 점프 | 높이 3~4타일 — 단순 점프로 충분 |
| 🌵 **Cactus** | 이중 점프 | 높이 12~13타일 — 단순 점프로 넘기 불가 |
| 🐦 **Bird** | 슬라이드 | 3마리 수직 배치 — 점프로 위를 넘을 공간 없음 |

> 장애물 크기와 배치가 의도된 회피 동작을 구조적으로 강제합니다.

## 조작법

| 키 | 동작 |
|----|------|
| `↑` / `W` / `Space` | 점프 (공중 재입력 시 이중 점프) |
| `↓` / `S` (누르는 동안) | 슬라이드 |
| `ESC` | 일시정지 |
| 터치 위로 스와이프 | 점프 |
| 터치 아래로 스와이프 | 슬라이드 |

## 실행 방법

빌드 없이 `index.html`을 브라우저에서 바로 열면 됩니다.

```bash
git clone https://github.com/ashes331/Special-Playground.git
cd Special-Playground/hero-runner
open index.html       # macOS
# xdg-open index.html  (Linux)
# start index.html     (Windows)
```

## 구현 메모

- **스프라이트** — 달리기 4프레임 / 점프 포즈 / 슬라이드 포즈를 `fillRect` 픽셀 블록으로 직접 구현, 외부 이미지 없음
- **배경 레이어** — `#bg-scene` 별도 캔버스를 `position: fixed`로 깔고, 게임 캔버스와 분리 렌더링
- **산 실루엣** — LCG 시드 난수로 봉우리 좌표 생성 후 `TILE(6px)` 단위 계단식으로 그려 픽셀 느낌 유지
- **파티클** — 점프 / 코인 획득 / 사망 시 유형별 색상과 크기로 분기
- **터치 지원** — `touchstart` / `touchend` 스와이프 방향으로 점프 / 슬라이드 매핑
